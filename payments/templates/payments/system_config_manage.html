How It Works
PayMongo Automatic Payment System

The registration fee is now fully automated using PayMongo. No more manual QR code uploads or receipt verification!

For Self-Registering Users:
  1. User creates an account
  2. System generates PayMongo payment link automatically
  3. User clicks "Pay Now" button
  4. User scans QR code with GCash or Maya
  5. Payment verified automatically via webhook (5-30 seconds)
  6. User account activated automatically
For Teacher Bulk Registration:
  1. Teacher creates multiple student accounts
  2. System calculates total (students × registration fee)
  3. Teacher pays total amount once via PayMongo
  4. All student accounts activated automatically when paid
Payment Methods:
  • GCash - via PayMongo QR code
  • Maya (PayMaya) - via PayMongo QR code
Zero Manual Work: All payments are verified automatically. No receipt uploads, no admin verification needed!
Note: Changing the registration fee will only affect NEW registrations. Existing pending payments will keep their original amount.
